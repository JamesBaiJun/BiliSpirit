## 🎞 演示：
[演示视频](https://www.bilibili.com/video/BV1od4y1g7xg)
## 💎 特别说明
> * **程序调用Blibili的Web端Api接口获取数据** 
> * **只允许扫描二维码登录** 
> * **仅显示视频信息，不提供本地播放的功能，点击视频将跳转Bilibili对应的网页** 
> * **程序支持置顶自动收起和侧边栏模式** 
> * **程序当前仅为初步测试版本，可以在Release中下载已编译的版本** 
> * **通过扫码登录获取的Cookie数据保存于程序路径下的Cookie.txt，删除该文件即可重新登录**
